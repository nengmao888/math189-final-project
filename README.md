# Math 189 Final Project

UCSD Math 189 Final Project: Statistical analysis of factors influencing product ratings

## Project Proposal Requirements

The proposal should be 1–2 pages (9–11pt font, single-spaced, PDF format).

It should address the following questions:

- Why is this problem important?
- What are the data sources?
- Where will the data be acquired?
- What types of analysis will be performed?
- What are the expected outcomes?

The proposal must be written in prose (not Q&A format).

## Grading Rubric

- Clarity (40%): Clear objectives and well-organized writing  
- Difficulty & Originality (40%): Challenging and meaningful project  
- Feasibility (10%): Realistic and achievable plan
